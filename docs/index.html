注意 / Notes
- 本リポジトリはデモ用途です。デモ画像の再利用・再配布は禁止です（DEMO_IMAGES_LICENSE.txt を参照）。
- This repository is for preview only. Demo images must not be reused or redistributed (see DEMO_IMAGES_LICENSE.txt).
- モデル重みは同梱していません。必要なモデルは各配布元（例：Civitai等）から入手してください。
- Model weights are not included. Please obtain required models from their distribution pages. 特記事項
- 本デモは各表情3回（最大3seed）だけ生成しています。ウインク/閉じ目笑顔など、一部は表情名どおりでない場合があります。
- This demo is limited to 3 runs (up to 3 seeds) per expression. Some results (e.g., wink / closed-eye smile) may not match the intended label.
- 試行回数を増やせば自然になりますが、プロンプト価値保護のため制限しています。
- More attempts would yield more natural results, but the limit is intentional to protect the value of the prompt presets.
- 不自然さが多い＝安定性が低い目安。眼鏡ありは特に崩れやすいです。
- More unnatural outputs indicate lower consistency. Glasses tend to break more easily.
